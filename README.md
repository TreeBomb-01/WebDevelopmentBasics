## 📘 1. 프로젝트 개요  
본 프로젝트는 **LevelUpFit**의 실제 서비스 구성을 기반으로 제작한  
AI 운동 서비스 랜딩 페이지 웹사이트이다.

총 3개의 페이지(확장 가능)로 구성되며,  
HTML/CSS/JS를 활용하여 **현대적인 UI**, **애니메이션**, **카드 레이아웃**,  
**다크모드 토글**, **슬라이더**, **폼 유효성 검사** 등을 포함한다.

---

## 🎯 2. 페이지 구조  

### ✔ 1) index.html – 메인 랜딩 페이지  
서비스 전체 소개 페이지로 구성하며 다음을 포함한다.

- Hero 섹션(대형 배너 + CTA 버튼 2개)  
- LevelUpFit 핵심 기능 소개(카드 3~6개)  
- 예시 루틴/AI 기능 이미지  
- 사용자 후기 슬라이더(JS)  
- 다크모드 토글  
- Footer(SNS 아이콘 포함)  

### ✔ 2) routine.html – 루틴 설명 페이지  
- LevelUpFit의 대표 루틴 3가지 이상  
- 난이도, 소요 시간, 준비물, 설명  
- 루틴 아코디언 펼치기/접기 기능(JS)  
- 이미지/아이콘 포함  

### ✔ 3) signup.html – 회원가입 안내 페이지  
- 심플한 회원가입 UI  
- 실시간 입력 검증(JS)  
- 시작하기 버튼 클릭 시 모달 또는 alert  
- 카메라 권한 안내 텍스트  

---

## 🧩 3. LevelUpFit 기반 기능 소개 예시  
index.html의 기능 소개 섹션에서 다음 기능을 사용한다.

1. **AI 자세 분석**  
   스마트폰 카메라로 실시간 자세 평가 및 피드백 제공  

2. **맞춤 운동 루틴 추천**  
   개인 체력·목표 기반 AI 추천 알고리즘  

3. **운동 기록 관리**  
   일일 운동 기록 자동 저장 및 분석  

4. **데일리 챌린지 시스템**  
   꾸준함을 위한 목표 기반 챌린지 제공  

5. **모바일 연동**  
   웹 · 모바일 모두 가능

## 🛠 5. 사용 기술

### HTML5  
- 시맨틱 태그(header, nav, section, article, footer)  
- 반복되는 UI 카드 구조  

### CSS3  
- Flexbox / Grid  
- 반응형 디자인  
- Hover 효과  
- Box Model 스타일링  
- 다크모드 테마  

### JavaScript  
- 다크모드 토글  
- 후기 슬라이더  
- 아코디언 인터랙션  
- 폼 유효성 검사  
- 간단한 페이드/스케일 애니메이션
