# WebDevelopmentBasics — LevelUpFit 랜딩 페이지

## 프로젝트 개요
- **프로젝트 성격**: 학술제에 직접 제작하여 출품한 프로젝트의 **랜딩 페이지(소개/시연 페이지)**
- **콘셉트**: "완벽한 운동 자세를 위한 AI 퍼스널 트레이너"를 주제로, 운동 영상 기반 **자세 분석/피드백 서비스**의 핵심 가치와 사용 흐름을 한 페이지에서 전달

## 페이지에서 전달하는 핵심 메시지
- 영상 기반으로 관절 위치를 추적하고, 난이도(3단계) 기준으로 자세를 분석한다는 컨셉
- 혼자 운동할 때 발생할 수 있는 부상 위험을 줄이고, 정확한 운동 폼 학습을 돕는다는 목적

## 랜딩 페이지 구성(섹션)
- **Hero**: 서비스 한 줄 소개 + 주요 CTA(체험 섹션으로 스크롤)
- **App Preview**: 기능별 시연 영상 전환(미리보기)
- **Stats**: 핵심 성과/지표 요약
- **Professional AI Analysis(Features)**: 정밀 모션 인식 / 3단계 난이도 평가 / 실시간 교정 피드백
- **How It Works(Process)**: 촬영·업로드 → AI 분석 → 리포트 확인
- **AI 분석 체험(Demo)**: 난이도 선택 + 영상 업로드 UI + 리포트 영역
- **FAQ**: `<details>/<summary>` 기반 아코디언

## 기술 스택
- **HTML / CSS / Vanilla JavaScript**
- 아이콘: **Font Awesome**(CDN)

## 폴더 구조
```
.
├─ index.html
├─ css/
│  └─ index.css
├─ js/
│  └─ main.js
├─ images/
└─ videos/
```

## 실행 방법
- 별도 빌드/설치 없이 `index.html`을 브라우저로 열면 됩니다.
- 동영상 미리보기는 `videos/` 폴더 내 파일을 사용합니다.

## 비고
- 본 저장소는 "서비스 소개/시연"을 목표로 한 **프론트엔드 랜딩 페이지**이며, 실제 AI 분석 모델/서버는 포함하지 않습니다.
- **AI 분석 체험(Demo)** 영역은 실제 서비스 연동 없이도 화면을 확인할 수 있도록, 예시 리포트(샘플 데이터)가 표시되게 설정해 두었습니다.
